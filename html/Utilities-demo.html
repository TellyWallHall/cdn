Math & Numbers

Clamp

Restrict a value between min and max bounds

Utilities.clamp(15, 0, 10)
Lerp (Linear Interpolation)

Interpolate between two values

Utilities.lerp(0, 100, 0.5)
Random Range

Generate random numbers in a range

Utilities.randomRange(1, 10)
Map Range

Map a value from one range to another

Utilities.mapRange(5, 0, 10, 0, 100)

Arrays & Collections

Shuffle

Randomly shuffle an array

Utilities.shuffle([1, 2, 3, 4, 5])
Chunk

Split array into chunks of specified size

Utilities.chunk([1, 2, 3, 4, 5, 6], 2)
Unique

Remove duplicates from array

Utilities.unique([1, 2, 2, 3, 4, 4, 5])
Group By

Group array items by property

Utilities.groupBy(users, 'age')

Truncate
